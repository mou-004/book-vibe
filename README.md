# 📚 Book Vibe

A modern book discovery and reading tracker web application built with React. Book Vibe allows users to explore books, manage their reading journey, and visualize progress in a clean and interactive way.

---

## 🚀 Features

* 📖 Browse a curated collection of books with detailed information
  (category, tags, rating, publisher, and more)

* ✅ Add books to **Read List** or **Wishlist**

* 🔄 Automatically move books from Wishlist → Read List

* 📊 Sort books by:

  * Rating
  * Number of pages
  * Publishing year

* 📈 Visualize reading progress using **Recharts**

* 🔔 Toast notifications for user actions

* 🌗 Light & Dark mode support (DaisyUI)

* 💾 Persistent data using **Local Storage**

* 📱 Fully responsive design (mobile, tablet, desktop)

---

## 🛠️ Tech Stack

* ⚛️ React 19
* ⚡ Vite
* 🎨 Tailwind CSS v4
* 🌼 DaisyUI v5
* 🔗 React Router v7
* 📊 Recharts
* 🔔 React Toastify
* 🎯 Lucide React Icons
